# LIT | Executive Learning Analytics Dashboard

This tool is part of the **Leadership Insight Tools (LIT)** portfolio.

## Project Description

The Executive Learning Analytics Dashboard is designed for C-suite leaders to visualize the business impact of leadership development programs. It connects simulated data from a Learning Record Store (LRS) with business KPIs to provide actionable insights.

## Core Functionality

*   **KPI Visualization:** Real-time display of Total Investment, Pipeline Impact, Average Competency Gain, and Program ROI.
*   **Competency Improvement Chart:** Visualizes "before and after" competency levels across the leadership team.
*   **Detailed Learner Data:** A granular view of individual progress, development areas, and assessment scores.

## Showcase Goals

*   **Learning Analytics:** Translating learning data into business value.
*   **Data Visualization:** Creating clear, executive-ready narratives from complex data.
*   **React Development:** Demonstrating proficiency with modern React and charting libraries.

## How to Run the Application

1.  **Navigate to the project directory:**
    ```bash
    cd analytics-dashboard
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Start the development server:**
    ```bash
    npm run dev
    ```
