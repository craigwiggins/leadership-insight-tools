# Leadership Insight Tools (LIT)

Welcome to the **Leadership Insight Tools (LIT)** portfolio. This suite of applications, developed by Craig Wiggins, showcases the intersection of executive education, learning analytics, and performance-driven instructional design using modern web technologies.

## Portfolio Overview

LIT demonstrates how data and AI can be leveraged to move beyond traditional "course-based" learning into strategic, real-time leadership development.

### 1. [AI-Powered Competency Architect](./competency-architect/)
A strategic tool that uses Gemini AI to generate personalized, blended learning curricula based on selected executive competencies. It links recommendations to instructional design principles for a high-impact development experience.

### 2. [Executive Learning Analytics Dashboard](./analytics-dashboard/)
A C-suite visualization platform that connects Learning Record Store (LRS) data with business KPIs. It provides clear insights into competency gains, program progress, and estimated ROI.

### 3. [xAPI-Powered Performance Support Tool](./xapi-support-tool/)
A "just-in-time" support system that tracks user behavior in a workflow (via xAPI) and provides contextual help resources exactly when they are needed, demonstrating a commitment to performance improvement.

## Showcase Goals

*   **Executive Education:** Crafting development plans for senior leaders.
*   **Strategic Curriculum Design:** Architecting complex, multi-faceted learning programs.
*   **Performance Improvement:** Focusing on learning that drives measurable outcomes.
*   **Learning Analytics:** Translating raw data into meaningful business insights.
*   **xAPI & TLA:** Practical application of the Experience API and Total Learning Architecture.
*   **AI in Learning:** Applying AI for sophisticated personalization and instructional scaling.

## Technical Stack

*   **Frontend:** React (JavaScript/TypeScript)
*   **Build Tool:** Vite
*   **AI Integration:** Google Gemini API
*   **Data Standards:** xAPI (Experience API)
*   **Visualization:** Chart.js / Recharts (as seen in the dashboard)
*   **Styling:** CSS3 / Modern UI principles
